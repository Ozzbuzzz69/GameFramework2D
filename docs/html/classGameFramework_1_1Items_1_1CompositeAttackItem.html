Detailed Description

Composite attack item that aggregates a collection of IAttackItem instances.

Hit and Weight return the sums of all children; Range returns the maximum child range.

Implements IEnumerable<T> so collection-initialiser syntax works:

var combo = new CompositeAttackItem("Battle Kit") { sword, axe, magicStaff };
