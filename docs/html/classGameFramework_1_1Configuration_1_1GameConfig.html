Detailed Description

Reads and stores game configuration from an XML file.

Expected XML format:

&lt;GameConfig&gt;
&lt;World MaxX="20" MaxY="20" /&gt;
&lt;Level&gt;Normal&lt;/Level&gt;
&lt;/GameConfig&gt;
