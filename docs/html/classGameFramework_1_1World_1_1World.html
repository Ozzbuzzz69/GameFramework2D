Detailed Description

Represents the 2D game world.

Holds all Creature and WorldObject instances and enforces boundary rules.

World size is loaded from Configuration.GameConfig:

var config = GameConfig.Load("game.xml");
var world = new World(config.MaxX, config.MaxY);
